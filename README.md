# Podsite1

A simple Next.js podcast site.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the app in your browser:
   ```bash
   http://localhost:3000
   ```

## Build

To create a production build:

```bash
npm run build
```

## Start Production

After building, run:

```bash
npm start
```
